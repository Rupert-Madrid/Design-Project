These are the codes used for Design of a Severity Classifier Model with Recommendation System for Analyzing Road Accidents

Collision Detection uses a yolov5 model to detect road accidents on camera. Said accidents are then analyzed further through the resnet50 classification model to determine severity before details are sent to the Web Application.
